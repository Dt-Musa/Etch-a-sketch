## ğŸš€ Getting Started

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dt-Musa/Etch-a-sketch.git
   cd Etch-a-sketch
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python

     # Using Node.js
     ```

   - Hover over the grid to create your masterpiece!
